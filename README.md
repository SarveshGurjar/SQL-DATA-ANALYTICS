# SQL-DATA-ANALYTICS

SQL practice for data analytics — queries on real datasets, from basic to advanced scenarios.

This repository contains **SQL practice projects** using a sales and employee dataset to answer business questions and simulate real-world analytics tasks.

---

## Basic SQL Concepts

- **Dataset:** Sales dataset
- **Concepts Covered:** SELECT, WHERE, GROUP BY, JOINS
- **Purpose:** Learn to write queries to answer fundamental business questions like sales totals, product trends, and customer analysis.

---

## Advanced SQL – Employee Performance Analytics

This project demonstrates **advanced SQL concepts** used in real-world data analytics environments.

### Concepts Covered
- Relational data modeling  
- CTEs (WITH clauses)  
- Window functions (LAG, LEAD, ROW_NUMBER)  
- Business metrics (salary growth %, performance trends)  
- Analytics-layer table design and reporting

### Example Use Cases
- Analyze employee salary growth and trends  
- Evaluate promotion alignment across departments  
- Detect performance anomalies using window functions  
- Aggregate business metrics for executive dashboards

### Tools Used
- MySQL  
- VS Code  
- GitHub

## Skills Demonstrated

- Writing **scalable and efficient SQL queries**  
- Translating business questions into **actionable data queries**  
- Using **analytical functions and aggregations** for insights  
- Handling **realistic datasets** and simulating professional scenarios
